# Prerequisites to run project

- Install nodejs
- Install yarn pkg
- Setup IDE

[Nodejs installer](https://nodejs.org/en/download/)

## Yarn package manager

[Yarn installer](https://classic.yarnpkg.com/lang/en/docs/install/)
